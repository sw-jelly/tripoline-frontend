# Tripoline
```
     어릴 때 즐겁게 타던 트램펄린처럼
     신나는 여행을 돕는 서비스!
```
SSAFY 1학기 프로젝트 (2023.09 ~ 2023.11)

### 참여 페어
- 지인성(조장), 양유경

### 성과 
SSAFY 10기 OO반 최종프로젝트 최우수상 수상
